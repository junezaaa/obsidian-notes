	 Midpoint (ค่ากลาง) is like normal value or representative of the whole dataset

- Metric to measure the midpoint
	- Mean/Average
	- Median
	- Mode

- Mean : you know
	- Use for numerical data
	- sensitive to outliers
-  Median มัธยฐาน : Middle value in a list of number
	- ex 1,2,3 median = 2
- Mode ฐานนิยม : Most frequent data
	- Usable for all dtype
