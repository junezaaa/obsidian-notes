
### Correlation สหสัมพันธ์
- How much relation between 2 variable in a statistical way
	- If x change how will y change
	- If the corr is high it can be use to create a feature
- Metric
	- Pearson is the corr a straight line (continous num)
		-  -1  x up y up 
		- 1 x down y down
		- 0 who gaf
	- Spearman - is the corr in the same direction? (ranking)
		- -1 if they go in a opposite way
		- 1 if they go in the same way
		- 0 is they have no corr
