[[Logical vs Physical Address Space|Logical addr]] generated by CPU is divided into:
- **Page number (p)** - used as an index into a [[Paging System|page table]] which contains base addr of each [[Simple Paging#^668812|page]] in physical memory.
- **Page offset (d)** - combined with [[Base and Bound Registers|base addr]] to define the [[Logical vs Physical Address Space|physical mem addr]] that is sent to the [[MM Unit|memory unit]]
